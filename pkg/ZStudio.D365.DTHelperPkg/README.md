# DTHelper - ZD365.DeploymentHelper

- DTHelper - ZD365.DeploymentHelper is a command line tool designed to assist with various deployment tasks in Dynamics 365 Customer Engagement (D365 CE).
- This nuget package contains the compiled binaries for DTHelper and should be added into a deployment project.
- The package will extract all the assemblies into a folder named 'ZD365DTHelper' on the project.
- For more information about this package, go to the [DTHelper GitHub Repository](https://github.com/zerox2cool/d365dt).
